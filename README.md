## Brief

Reactive and responsive E-commerce website using Noroff API.

## Built With

React, JS

## Getting Started

### Installing

1; Clone the repo:

```bash
git clone https://github.com/MegumiKim/lemarche.git
```

2; install dependencies

```bash
npm i
```

### Running

```bash
npm run build
```
